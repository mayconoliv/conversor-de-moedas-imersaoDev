## Projeto

 **Projeto desenvolvido para a Alura, Imersão.Dev**

 O projeto tem como objetivo desenvolver um conversor de moedas, onde o usuário insere o valor de uma das moedas (dólar, euro, bitcoin ou iene) e o valor é convertido para o Real.
